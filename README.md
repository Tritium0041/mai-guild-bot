# mai-guild-bot
提供maimai数据查询服务的qq频道bot  
在运行代码之前，您需要从[此链接](https://www.diving-fish.com/maibot/static.zip)下载资源文件并解压到src文件夹中。  
下载后请自行申请qq机器人的appid和token  
可以使用screen直接运行
